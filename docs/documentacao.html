Documentação

Vamos aprender a usar esse framework em nossos projetos?

O MiniFramework agora está na sua primeira versão estável, aproveite os novos recurso.

Primeiros passos

Inicialização

Para começar baixe o MiniFramework ou crie um novo projeto.

Se você criar um novo projeto você terá uma estrutura de pastas pronta com os principais arquivos de um projeto, você só precisa baixar, compilar o Sass e iniciar o seu código html.

Baixar Novo projeto

Se você baixou somente o MiniFramework terá que incluí o css no <head> de seu documento html.

<link rel="stylesheet" href="./lib/miniframework/css/miniframework.min.css">

E também o javascript no final do <body> do seu documento html.

<script src="./lib/miniframework/js/miniframework.js"></script>

Cores

O MiniFramework vem com 8 cores, são elas: dark, light, white, gray, blue, green, red, yellow.

Para definir uma cor em um texto basta inserir a classe com o nome da cor desejada.

Para definir uma cor de background basta inserir a classe bg- e o nome da cor desejada.

bg-dark
bg-light
bg-gray
bg-white
bg-blue
bg-green
bg-red
bg-yellow

Elementos html

Loading

Coloque o loading em todas as suas páginas html, isso vai deixar seu site bem legal.

<div class="loading">
  <div class="loader"></div>
</div>

Sidebar

Para criar um menu lateral insira esse código em seu html.

<div class="sidebar-overlay"></div>

<nav class="sidebar bg-light">
  <header>
    <h4>Menu</h4>
    
    <button>
      <i class="bx bx-x"></i>
    </button>
  </header>

  <div>
    <a href="#" class="item active">Início</a>
    <a href="#" class="item">Serviços</a>
    <a href="#" class="item">Sobre</a>
    <a href="#" class="item">Contato</a>
  </div>
</nav>

Modal

Para criar um modal, em seu script, acesse o método set do objeto modal, no primeiro parâmetro indique o id do modal e no segundo parâmetro indique a class do botão que irá abrir esse modal.

Você poderá ter vários botões para abrir o mesmo modal.

modal.set('#modal', '.btn-modal')

Em seguida, crie o modal em seu html.

<div class="modal-overlay">
  <button class="close">
    <i class="bx-bx-x"></i>
  </button>
</div>

<section id="modal" class="modal">
  <div>
    <h2>Lorem ipsum dolor</h2>
    <p>Lorem ipsum, dolor sit amet consectetur, adipisicing elit. Ullam, eaque hic omnis fuga nam similique harum, magnam natus blanditiis reiciendis nostrum sequi ad, qui ipsam amet eveniet, modi ducimus repellat.</p>
  </div>

  <footer>
    <a href="#" class="btn">Ver mais</a>
  </footer>
</section>

Scroll to top

Um botão para voltar ao topo da página é muito útil, principalmente se a página for grande.

  <i class="bx bx-chevron-up"></i>
</button>
