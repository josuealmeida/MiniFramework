Um framework prático e produtivo

Esse framework foi desenvolvido pensando na praticidade, possuindo classes css com as propriedades mais usadas para o desenvolvimento diário.

Por que usar MiniFramework?

  • Muito fácil e prático.
  • Classes com os estilos mais usados no dia a dia.
  • Muito leve.
  • Totalmente responsivo.
  • Design moderno.
  • Vanilla js, sem dependências.
  • Informações e ajudas no console do seu navegador.
  • Você pode desenvolver uma página inteira utilizando pouquíssima codificação css.
  • Sem tempo? sem problema.
